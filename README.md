# Task Manager App

A lightweight and efficient Task Manager built as a front-end development project. 

## Key Features
- **Interactive UI:** Dynamic task addition and completion workflow.
- **Responsive Design:** Clean, modern interface optimized for desktop and mobile.
- **Frontend Logic:** Built using vanilla JavaScript to handle DOM manipulation and task states.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to use
Simply enter a new task, click "Add", and manage your workflow by marking tasks as done!
